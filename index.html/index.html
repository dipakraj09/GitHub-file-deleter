🔑 GitHub Token Setup Guide

Token generate karo aur sahi permissions set karo

Step 1 — GitHub Settings Kholo
1. github.com pe jaao
2. Profile photo click karo (top-right)
3. Settings click karo
4. Left sidebar mein sabse neeche Developer settings
5. Personal access tokens → Tokens (classic)
Step 2 — Naya Token Generate Karo
1. Generate new token (classic) click karo
2. Note mein naam likho: GitXplore
3. Expiration — No expiration choose karo
Step 3 — Yeh Permissions Check Karo ✅
☑ repo — Full control of private repositories ☑ repo:status ☑ repo_deployment ☑ public_repo ☑ repo:invite ☑ security_events ☑ delete_repo — Delete repositories (files ke liye zaroori) ☑ workflow — Update GitHub Action workflows
Step 4 — Token Copy Karo
1. Generate token button dabao
2. Token copy karo (sirf ek baar dikhega!)
3. Yahan Token field mein paste karo
⚠️ Token kabhi share mat karo — chat, screenshot, ya kisi ko bhi mat batao. GitHub automatically revoke kar deta hai jab token public ho jaata hai.
